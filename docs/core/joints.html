类型自由度说明

关节 API

C 函数说明
joint_builder_create创建关节构建器
joint_builder_set_limits设置平移/旋转限位

world_insert_impulse_joint插入关节
world_remove_impulse_joint移除关节

C 函数说明

controller_set_snap_to_ground启用地面对齐
