刚体操作

刚体是模拟中的基本动力学对象。每个刚体具有质量属性、位姿 (位置+旋转)、速度 (线速度+角速度)，可接受力和冲量。

02

构建器 API

C 函数说明
rigid_body_builder_create创建构建器 (Dynamic/Fixed/Kinematic)
rigid_body_builder_set_translation设置初始位置
rigid_body_builder_set_rotation设置初始旋转 (四元数)
rigid_body_builder_set_pose同时设置位置+旋转
rigid_body_builder_set_linvel设置初始线速度
rigid_body_builder_set_angvel设置初始角速度
rigid_body_builder_set_additional_mass设置点质量 (kg)
rigid_body_builder_set_additional_mass_properties设置完整质量属性
rigid_body_builder_set_linear_damping设置线速度阻尼
rigid_body_builder_set_angular_damping设置角速度阻尼
rigid_body_builder_set_ccd_enabled启用连续碰撞检测
rigid_body_builder_build构建刚体

rigid_body_set_angvel设置角速度

力/冲量

C 函数说明

Rigid Body

Rigid bodies are the fundamental dynamic objects. Each has mass, pose, velocity, and can receive forces/impulses.

02

rigid_body_builder_set_poseSet position + rotation at once
rigid_body_builder_set_linvelSet initial linear velocity
rigid_body_builder_set_angvelSet initial angular velocity
rigid_body_builder_set_additional_massSet point mass (kg)
