世界管理

PhysicsWorld 是模拟的核心容器，拥有所有刚体、碰撞体、关节、力注册表、事件系统和共享内存 Arena。单一入口：仅需世界指针即可操作所有子系统。

快速开始： world_create(g) → world_step(w, dt) → world_destroy(w)。三行代码即可运行完整物理模拟。

01

world_destroy✅销毁世界及所有资源

world_update_body_poses✅批量更新刚体位姿
world_update_body_velocities✅批量更新刚体速度
world_register_celestial_gravity✅注册天体引力到 ForceRegistry
world_get_force_registry_count✅已注册力律数量

积分参数

参数默认值说明
dt1/60时间步长 (秒)
num_solver_iterations4每个子步的约束求解器迭代次数
max_ccd_substeps1快速移动体的 CCD 子步数

生命周期

// 创建 → 配置 → 模拟 → 销毁
let world = world_create(Vec3 { x: 0.0, y: -9.81, z: 0.0 });
world_set_integration_parameters(world, 1.0/120.0, 8, 2);
// ... 插入刚体、碰撞体、注册力 ...
loop { world_step(world, dt); }
world_destroy(world);

World Management

PhysicsWorld is the central simulation container owning all bodies, colliders, joints, force registry, event system, and shared memory arena.

01
