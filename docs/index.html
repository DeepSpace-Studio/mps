核心特性

高精度引力

球谐展开 (EGM2008 8×8)、椭球引力、J2-J6 带谐、四极张量。自动根据轨道高度选择最优模型。

辛积分器

Leapfrog、Yoshida 4 阶、Forest-Ruth 8 阶。Kahan 补偿精度从 15 位→30 位有效数字。后牛顿 1PN+2PN 相对论修正。

内置天体

太阳系 10 天体精密参数 (JPL DE441)。地球 EGM2008、月球 LP165 + 12 Mascon (GRAIL)、火星 Mars50c。

地形引力

多面体引力 (Werner-Scheeres)、DEM 地形质量分布、FFT 加速。月球 Mascon 模型防止低轨坠毁。

ForceRegistry

类型化力注册表。任意力实现 ForceLaw trait 后自动调度，世界步进内自动聚合报告，无需手写分发逻辑。

JNI + 共享内存

Java 21 JNI 全绑定 (~280 方法)。共享内存 Arena (DirectByteBuffer) 零 JNI 读写，每帧仅 1 次 world_step 调用。锁自由事件系统。

Core Features

High-Precision Gravity

Spherical harmonics (EGM2008 8×8), ellipsoidal gravity, J2-J6 zonal, quadrupole tensor. Auto-selects optimal model by orbital altitude.

Symplectic Integrators

Leapfrog, Yoshida 4th, Forest-Ruth 8th. Kahan compensated summation: 15→30 significant digits. Post-Newtonian 1PN+2PN corrections.

Built-in Celestial Bodies

10 solar system bodies (JPL DE441). Earth EGM2008, Moon LP165 + 12 Mascon (GRAIL), Mars Mars50c.

Terrain Gravity

Polyhedron gravity (Werner-Scheeres), DEM mass distribution, FFT acceleration. Lunar Mascon model for low-orbit safety.

ForceRegistry

Typed force registry. Any force implementing ForceLaw trait auto-dispatches during world step with aggregated reporting.

JNI + Shared Memory

Java 21 JNI bindings (~280 methods). Shared memory Arena (DirectByteBuffer) with zero-JNI read/write. Lock-free event system.
