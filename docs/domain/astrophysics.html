N 体引力模拟 (3 种算法)

函数算法复杂度最佳场景
nbody_direct_sum直接 O(N²) 求和O(N²)N < 1000，高精度需求
nbody_barnes_hutBarnes-Hut 树算法O(N log N)N 1000–100000，中等精度

N-Body Gravity (3 Algorithms)

FunctionAlgorithmComplexityBest For
nbody_direct_sumDirect O(N²) sumO(N²)N < 1000, high precision
nbody_barnes_hutBarnes-Hut tree algorithmO(N log N)N 1000–100000, medium precision
nbody_fmm_monopoleFast Multipole Method monopoleO(N)N > 100000, large-scale sims

All three algorithms have configurable precision. Barnes-Hut uses θ to control expansion accuracy (θ→0 is equivalent to direct sum). FMM currently implements monopole acceleration only; higher multipole orders are planned.

Post-Newtonian Correction

FunctionDescription
post_newtonian_correction1PN + 2PN post-Newtonian acceleration correction

In strong gravitational fields (Mercury, binary systems), Newtonian gravity needs relativistic corrections. The 1PN term produces Mercury's perihelion precession (~43"/century); the 2PN term describes gravitational wave radiation damping.
